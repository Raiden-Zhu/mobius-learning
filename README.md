# Möbius Learning

This repository contains the static project page and approved public-release assets for “Möbius Learning: Cyclic Depth Folding in Transformers.” It is intentionally separate from the research working repository.

## Local preview

Serve the `docs/` directory from the repository root:

```bash
python3 -m http.server 8000 --directory docs
```

Then open `http://localhost:8000/`.

## GitHub Pages

The public repository is `Raiden-Zhu/mobius-learning`. GitHub Pages is served from the `main` branch and `/docs` directory at `https://raiden-zhu.github.io/mobius-learning/`, with HTTPS enforced.

The complete report PDF and LaTeX sources are not stored in this repository. Paper links remain disabled until an approved external paper URL is available.
